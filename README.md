# 🧠 AI Health Risk Assessment System (V1.0)

A web-based health assessment system that evaluates users’ health risks based on lifestyle and personal data.

Built with Python and Streamlit, this project provides real-time feedback, visual analytics, and historical tracking.

---

## 🚀 Features

- 📊 Health Score Calculation (Rule-based system)
- 🟢 Risk Level Classification (Low / Medium / High)
- 📏 Risk Progress Bar
- 🕸️ Radar Chart Visualization (multi-dimension health analysis)
- 🧾 Structured Health Report (summary + recommendations)
- 📌 Key Indicators breakdown (BMI, Sleep, Diet, Activity, Hydration)
- 🗄️ Data Storage with MySQL
- 📈 History Tracking & Trend Analysis
- ⏳ Loading interaction for better UX

---

## 🧠 How It Works

The system collects user input such as:

- Personal information (height, weight)
- Lifestyle habits (diet, exercise, sleep)
- Mental health indicators

It then:

1. Calculates risk scores for each category  
2. Aggregates into an overall health score  
3. Classifies risk level  
4. Generates structured feedback and recommendations  
5. Visualizes results using charts and progress indicators
